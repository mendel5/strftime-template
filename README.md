# strftime-template
Templates to format date and time, e.g. on Linux systems

German:
%A _ %Y-%m-%d _ %j. Tag _ UTC %z _ %H:%M:%S Uhr
%A, %d. %B %Y, %j. Tag, %Z, UTC %z, %H:%M:%S Uhr

English:
%A _ %Y-%m-%d _ day %j _ UTC %z _ %H:%M:%S
%A, %d. %B %Y, day %j, %Z, UTC %z, %H:%M:%S
